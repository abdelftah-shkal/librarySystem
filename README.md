# Library System

A Java-based library management system for managing books, users, and library operations.

## Features

- Book catalog management
- User account management
- Loan and return tracking
- Library inventory management

## Project Structure

```
librarySystem/
├── src/              # Source code directory
├── .gitignore        # Git ignore file
└── librarySystem.iml # IntelliJ project file
```

## Getting Started

### Prerequisites

- Java 8 or higher
- IntelliJ IDEA or any Java IDE

### Installation

1. Clone the repository:
```bash
git clone https://github.com/abdelftah-shkal/librarySystem.git
```

2. Open the project in your IDE
3. Build and run the project
